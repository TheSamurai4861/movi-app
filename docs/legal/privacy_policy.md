## Politique de confidentialité — Movi

Dernière mise à jour : 2026-03-31

Movi est une application de lecture et de gestion de bibliothèque média. Movi **ne fournit pas de contenu** : l’utilisateur ajoute ses propres sources (BYO, « bring your own source »).

## 1) Données collectées

Selon les fonctionnalités utilisées, Movi peut traiter les catégories de données suivantes :

- **Compte / authentification (optionnel)** : identifiant de compte (ex. email) si vous vous connectez.
- **Profils** : nom/alias de profil, préférences d’application (langue, thème, etc.).
- **Préférences de lecture** : paramètres liés à la lecture (qualité préférée, langues audio/sous-titres, etc.) lorsque ces options sont activées dans l’app.
- **Historique de lecture** : progression (position de reprise), contenu regardé, épisodes/saisons.
- **Sources IPTV BYO** : informations nécessaires pour se connecter à vos sources (par ex. URL/endpoint). **Les identifiants restent sous votre contrôle** et sont stockés localement de manière sécurisée lorsque possible.
- **Rapports de problème** : si vous signalez un souci, le rapport peut inclure une description et des métadonnées liées au contenu concerné.

## 2) Finalités

Movi traite ces données pour :

- Fournir les fonctionnalités de base (lecture, bibliothèque, profils).
- Synchroniser votre bibliothèque/historique **si** vous activez des fonctions cloud.
- Améliorer la stabilité (diagnostic d’erreurs) et répondre aux demandes support.

## 3) Partage avec des tiers

Movi peut communiquer avec des services tiers strictement nécessaires :

- **Fournisseurs de paiement** (Apple/Google/Microsoft) : pour gérer les achats intégrés (abonnements), si vous souscrivez.
- **Backend cloud** (si activé) : pour l’authentification et/ou la synchronisation (ex. Supabase).
- **Fournisseurs de métadonnées** : pour récupérer des informations d’enrichissement (affiches, cast, etc.) si l’app les utilise.

Movi ne vend pas vos données personnelles.

## 4) Sécurité

- Les communications réseau utilisent le chiffrement quand disponible (HTTPS). Certaines sources IPTV BYO peuvent être en HTTP selon votre fournisseur ; dans ce cas, le trafic peut ne pas être chiffré entre votre appareil et votre source.
- Les données sensibles sont stockées localement de manière sécurisée lorsque la plateforme le permet (ex. Keychain/Keystore via stockage sécurisé).

## 5) Conservation

Les données sont conservées :

- Tant que nécessaire au fonctionnement (ex. préférences, historique).
- Jusqu’à suppression par l’utilisateur (suppression de profil, désactivation sync, etc.).

## 6) Vos choix et droits

Vous pouvez :

- Utiliser Movi en mode local si disponible.
- Modifier/supprimer vos profils et sources.
- Demander la suppression de vos données cloud liées à un compte (si vous êtes connecté).

## 7) Contenu et légalité (BYO)

Movi est un outil. Vous êtes responsable de vous assurer que vous disposez des droits/abonnements nécessaires pour accéder au contenu de vos sources.
