FAQ

Tout ce qu'il faut pour ajouter correctement votre première source.

À quoi sert le nom de la source ?

C'est un libellé local dans Movi. Il vous aide à reconnaître votre source plus tard, par exemple "IPTV Maison" ou "Abonnement principal".

Que faut-il mettre dans URL du serveur ?

Saisissez l'adresse complète fournie par votre service IPTV, généralement au format https://example.com:port. N'ajoutez pas de chemin supplémentaire si votre fournisseur ne le demande pas.

Username et mot de passe : où les trouver ?

Ces identifiants sont généralement fournis avec votre abonnement. Ils ne sont pas ceux de votre compte e-mail ou de votre compte Movi.

Pourquoi l'activation peut échouer ?

Les causes les plus courantes sont une URL incorrecte, un port manquant, un identifiant erroné ou un serveur temporairement indisponible. Vérifiez chaque champ avant de réessayer.

Mes identifiants sont-ils enregistrés ?

Oui, la source activée est conservée pour simplifier les ouvertures suivantes. Si une synchronisation cloud est disponible, elle peut aussi être sauvegardée à distance.

Puis-je ajouter plusieurs sources plus tard ?

Oui. Cet écran couvre seulement la première activation. Une fois dans l'application, vous pourrez gérer plusieurs sources si votre configuration le permet.
